How to Learn Javascript

For many of you, Javascript will be your first proper programming language. For others, some of this will seem familiar if you've ever worked in C/C++, Java or Python.

Learning this new language involves:

  • Building blocks: putting together many small pieces, figuring out how they fit together
  • Taking notes: how does [thing] work? When and why do I use it?
  • Keeping organized: keep your HTML in an HTML folder, CSS in a CSS folder, etc. Projects very quickly become huge and unwieldly. It's incredibly helpful to know where everything is. (No organized chaos.)
  • Hands-on experience: reading code is never a good substitute for trying things out on your own. Try re-running coding exercises, not just rereading them. You will retain more and for longer that way.

What exactly is Javascript?

If HTML is the content and CSS is the look and feel, where does Javascript fit in?

JS is what we use to do things. We can manupulate the browser's DOM (document object model), add new content, change styles, make complex animations, fetch and store data, or even make browser games!

In short:

HTML → content

CSS → style

JS → interactivity

    Conditionals

    An absolutely enormous amount of the code you will be writing will involve the use of conditionals, or statements that only execute conditionally. Their structure:

    if some condition is met, do something; else, do a different something.

    How does this look like in code?

    if ( condition ){
  // Do a thing or two or several
}
else {
  // Otherwise do something completely different
}

/* That condition can be anything that returns a boolean: */
a == b;
a !== b;
c > d;
c <= d;
e // e is a boolean

    This concept is absolutely critical! You will be using it extensively throughout the rest of this course.

    Arrays

    Variables can also be arrays, which are collections of other data. You can think of arrays as variables containing other variables.

    Syntax for declaring an array:

    var array = [ ham, spam, eggs ];

    Syntax for retrieving an element:

    array [ index ];

    Each element in an array has a number, starting from 0 and ending at lengthâ€“1.

    Your 0th index element is the first element in the array.

    Which element is this: animals [ 2 ] ?

    var animals = [ 'axolotl', 'emu', 'pangolin', 'ibex' ];

// Whitespace doesn't matter, so this might be neater:

var animals = [
  'axolotl',
  'emu',
  'pangolin',
  'ibex'
];

// We can nest arrays as well:

var odds_evens = [
  [ 0, 2, 4, 6, 8],
  [ 1, 3, 5, 7, 9]
];
