Recursion

Recursion is an interesting concept in computer science, based heavily on mathematical computer science proofs. Generally speaking, they consist of functions with a base case, or a condition in which the function should return. Otherwise, the function calls itself, most often with an argument it repeatedly edits.

A classic example of a recursive function is one that returns the factorial of a number:

// Declare a recursive function to print out
//  the factorial of a number
var factorial = (num) => {
  if(num <= 1)
    return 1;
  return num * factorial(num-1);
}

It may help to draw this out, so after I demo it I'll draw the different stack frames as we recurse through, then "unravel", as it were.

This activity is pretty large, so we will be working on Part I before the break and Part II after.

Over the course of this assignment you are going to put together a function which uses constructors and user input to create and manage a team of players.

Start out by creating a constructor function called Player with the following properties and methods:

  • name: player name
  • position: player's position
  • offense: value between 1 and 10 corresponding to skill on offense
  • defense: value between 1 and 10 corresponding to skill on defense
  • goodGame: increase either the player's offense or defense property based upon a coinflip
  • badGame: decrease either the player's offense or defense property based upon a coinflip
  • printStats: print all of the player's properties to the screen

Now, create a program which allows the user to create 3 unique players; 2 starters and a sub. It should take as user input the name, position, offense, and defense of each player.

Once all of the players have been created, print their stats.
