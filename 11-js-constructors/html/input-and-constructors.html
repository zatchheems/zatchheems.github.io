Recursion

Recursion is an interesting concept in computer science, based heavily on mathematical computer science proofs. Generally speaking, they consist of functions with a base case, or a condition in which the function should return. Otherwise, the function calls itself, most often with an argument it repeatedly edits.

A classic example of a recursive function is one that returns the factorial of a number:

// Declare a recursive function to print out
//  the factorial of a number
var factorial = (num) => {
  if(num <= 1)
    return 1;
  return num * factorial(num-1);
}

It may help to draw this out, so after I demo it I'll draw the different stack frames as we recurse through, then "unravel", as it were.
