Today's Session

Good morning, everyone! Welcome to yet another wonderful Saturday session. This class will be dedicated entirely to reviewing what we've learned in class thus far.

Recap

HTML/CSS

  1. Basic parts of an HTML document (<!DOCTYPE>, <head>, <body>)
  2. Inline, internal and external CSS stylesheets
  3. Relative paths to link assets
  4. #ids and classes for styling elements

Git

  1. Advantages of using Git and source control management
  2. Git bash/Terminal to perform basic Git commands
  3. git clone, git add, git commit, git push
  4. How branching works, checkouts and pull requests

Bootstrap/Mobile Responsiveness

  1. Using a pre-built CSS library
  2. Steps to include Bootstrap CSS in an existing website
  3. The process of dissecting a layout in the Bootstrap grid system
  4. General understanding of Bootstrap components (panels, Jumbotron, nav bar, etc.)
  5. General understanding of @media queries, <meta> viewport tag

JavaScript

  1. Understanding what purpose JS serves with regards to HTML/CSS
  2. Understanding syntax and usage of the following:
    1. Variables
    2. Arrays
    3. console.log, alerts, prompts, confirms
    4. Conditional statements (if()...else if()...else)
    5. for loops
    6. functions
    7. Objects
    8. Scope
    9. Methods (i.e. functions as properties)
  3. Organization of a JS file (global vars, functions)
  4. Event handlers
  5. Generating randomness

jQuery

  1. Purpose of using a JS library like jQuery
  2. jQuery selector syntax $(...)
  3. Capture button clicks using jQuery syntax
  4. jQuery methods to change, add, remove HTML
  5. Use JS and jQuery to change HTML in response to code
